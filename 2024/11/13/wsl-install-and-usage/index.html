WSL 安装与使用

一、WSL 概述

Windows Subsystem for Linux (WSL) 是微软开发的一个技术，允许用户在 Windows 操作系统上直接运行 Linux 命令行和一些基本的 Linux 应用程序。通过这种方式，用户可以在不切换到虚拟机或双系统安装的情况下，使用 Linux 工具和环境。

四、总结
